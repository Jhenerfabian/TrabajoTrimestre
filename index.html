IF / ELSE

Permite tomar decisiones en el programa. Ejecuta un bloque si la condición es verdadera y otro si es falsa.

  • Evalúa una condición booleana
  • Controla el flujo del programa
int edad = 20;

if (edad >= 18) {
    System.out.println("Mayor de edad");
} else {
    System.out.println("Menor de edad");
}
Decisiones

WHILE

Repite un bloque de código mientras una condición sea verdadera.

  • La condición se evalúa antes de ejecutar
  • Puede no ejecutarse ninguna vez
int i = 1;
while (i <= 3) {
    System.out.println(i);
    i++;
}
Repetición condicional

FOR

Se usa cuando conoces cuántas veces se debe repetir una acción.

  • Inicialización
  • Condición
  • Incremento
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
Contadores

SWITCH

Selecciona una opción entre múltiples casos posibles.

  • Evalúa una variable
  • Ejecuta el case coincidente
int opcion = 1;

switch (opcion) {
    case 1:
        System.out.println("Comida");
        break;
    case 2:
        System.out.println("Bebida");
        break;
    default:
        System.out.println("Inválido");
}
